# Reda Brothers Landscape — Pitch Website

A modern, single-page website designed for **Reda Brothers Landscape Inc.**, a premier hardscape and outdoor design company based in Flanders, NJ with 28 years in business.

**Live Preview:** [reda-brothers-pitch.vercel.app](https://reda-brothers-pitch.vercel.app)

---

## About the Client

Reda Brothers Landscape is a family-owned landscape and hardscape company serving Morris County and Northern New Jersey since 1997. Led by John Reda, they specialize in poolscapes, stone patios, retaining walls, outdoor kitchens, driveways, and full estate landscaping. BBB A+ rated with a 5.0-star review average.

## Features

- **Full-viewport hero** with auto-advancing slideshow and Ken Burns animation
- **Frosted glass navigation** that adapts between hero overlay and scrolled states
- **Custom SVG logo** — stacked stone mark reflecting their hardscape craft
- **Real project photography** from the client's portfolio across service cards
- **Interactive portfolio carousel** with horizontal scroll
- **Google Maps embed** for service area visualization
- **Contact form** with project type selector and phone formatting
- **Google & Yelp review buttons** integrated into the testimonials section
- **Scroll reveal animations** via Intersection Observer
- **Fully responsive** — optimized for desktop, tablet, and mobile (iPhone SE to 4K)

## Tech

Single `index.html` file. No build step, no dependencies.

- HTML5 / CSS3 / Vanilla JS
- Google Fonts (Lora + Figtree)
- Unsplash (stock hero/showcase images)
- Client's WordPress media library (service & portfolio photos)

## Deployment

Hosted on Vercel with GitHub auto-deploy.

```bash
vercel --prod
```
